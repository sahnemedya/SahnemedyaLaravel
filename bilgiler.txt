Sistem Bilgisi:
- Laravel 12.0
- Php 8.3

Proje Çalıştırma:

- Github'dan indirildikten sonra .env dosyası hazırlanmalı
- composer install komutu çalıştırılmalı
- npm install komutu çalıştırılmalı
- gerekli ise php artisan key:generate çalıştırılabilir
- php artisan migrate --seed ile veritabanı tabloları oluşturulur
- php artisan serve ile sistem ayağa kaldırılır.
- npm run build ile css ve js dosyaları derlenir. Eğer sitede css işlememişse npm run dev kullanabilirsiniz (çalıştıktan sonra terminali kapatmayın).
- domain.com/cms sayfasından panele ulaşabilirsiniz.

Proje içerisinde kullanılan birçok kütüphane (dropzone, datetimepicker,jquery vs.) npm ile kurulmustur
ve css ve js dosyalarının içerisinde import edilmiştir.


Özel Commadlar

online sunucuya gecirdikten sonra değişmesi gerekenler
# 1. TEMEL AYARLAR
APP_NAME="Firma Adı"
APP_ENV=production          # ← local'den production'a
APP_DEBUG=false            # ← true'dan false'a (GÜVENLİK)
APP_URL=https://www.firmaadi.com  # ← localhost'tan gerçek domain'e

# 2. DATABASE (cPanel bilgileri)
DB_CONNECTION=mysql
DB_HOST=localhost          # Çoğu hosting'de localhost
DB_PORT=3306              # Standart MySQL port
DB_DATABASE=cpanel_username_dbname   # cPanel'de oluşturduğun database
DB_USERNAME=cpanel_username_dbuser   # cPanel database user
DB_PASSWORD=cpanel_db_password       # cPanel database şifresi

# 3. CACHE & SESSION (Shared Hosting için)
CACHE_DRIVER=file         # Redis yoksa file kullan
SESSION_DRIVER=file       # Database yerine file
QUEUE_CONNECTION=sync     # Hosting queue desteklemiyorsa sync

# 4. LOG AYARLARI
LOG_CHANNEL=single        # daily yerine single (disk tasarrufu)
LOG_LEVEL=error          # debug yerine error (performans)
LOG_DEPRECATIONS_CHANNEL=null  # Deprecation log'ları kapat

# 5. GÜVENLİK
APP_KEY=base64:....       # Mevcut key'ini koru, değiştirme!
-------------------------Güvenlik Ayarları:
# MAIL (İletişim formu için)
MAIL_MAILER=smtp
MAIL_HOST=mail.firmaadi.com    # Hosting mail server
MAIL_PORT=587                  # veya 465 (SSL)
MAIL_USERNAME=info@firmaadi.com
MAIL_PASSWORD=mail_password
MAIL_ENCRYPTION=tls           # veya ssl
MAIL_FROM_ADDRESS=info@firmaadi.com
MAIL_FROM_NAME="${APP_NAME}"

# BROADCASTING (Gereksizse kapat)
BROADCAST_DRIVER=log

# FILESYSTEM (Public uploads için)
FILESYSTEM_DISK=public
